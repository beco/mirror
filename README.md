Notion Mirror
=============

A php way to read Notion pages and upload resources to an s3 bucket.

# Operation

## 1. prepare notion


## 2. check for local variables needed

These `env` vars are mandatory:
- `notion_token`
- `env`

### Optional
- `AWS_ACCESS_KEY_ID`
- `AWS_SECRET_ACCESS_KEY`
